# TypeScript Ödev

Bu repo, **Mobil Programlama** dersi için hazırladığım **TypeScript ödevimdir**.  
Her soru ayrı bir dosyada yer alıyor (Soru1.ts, Soru2.ts, ... Soru20.ts).

---

 Soruların Açıklamaları

- **Soru1.ts:** Generic fonksiyon örneği. İki farklı tipte diziyi alıp tek bir dizide birleştiriyor.  
- **Soru2.ts:** Type Guard örneği. Fonksiyon aldığı değerin tipini kontrol ediyor.  
- **Soru3.ts:** Class (sınıf) yapısı ve erişim belirleyicileri (public, private, protected) örneği.  
- **Soru4.ts:** keyof kullanılarak obje özelliklerine erişilen örnek.  
- **Soru5.ts:** Fonksiyon aşırı yükleme (overloading) örneği.  
- **Soru6.ts:** Generic sınıf (MemoryCache) örneği. Veriyi ekleme, getirme ve temizleme işlemleri yapıyor.  
- **Soru7.ts:** Partial ve Readonly tiplerinin kullanıldığı kullanıcı güncelleme örneği.  
- **Soru8.ts:** Rest parametreleriyle toplanan sayıları hesaplayan fonksiyon.  
- **Soru9.ts:** Soyut sınıf (abstract class) ve ondan türeyen alt sınıf örneği (örnek: Circle).  
- **Soru10.ts:** Statik özellik ve metot örneği (örnek: MathHelper sınıfı).  
- **Soru11.ts:** Conditional types (koşullu tipler) örneği. Promise içindeki değeri çıkarıyor.  
- **Soru12.ts:** infer ile fonksiyonun dönüş tipini bulan örnek.  
- **Soru13.ts:** Mapped types ve template literals ile get metotları oluşturan örnek.  
- **Soru14.ts:** Recursive tip örneği. DeepReadonly kullanılarak nesne içi özellikler sabitleniyor.  
- **Soru15.ts:** Mapped types ile sadece belirli tipteki özellikleri seçen örnek.  
- **Soru16.ts:** Nominal (branded) tip örneği. Farklı ID tiplerinin karışmasını engelliyor.  
- **Soru17.ts:** Union tiplerinden belirli değerleri filtreleyen örnek.  
- **Soru18.ts:** infer ile bir fonksiyonun son parametresinin tipini bulan örnek.  
- **Soru19.ts:** infer ile dizinin eleman tipini bulan örnek.  
- **Soru20.ts:** Template literal parsing örneği. URL içindeki parametreleri ayıklıyor.
